# Milestone v0.40: Pre-Beta Refactoring

**Status:** ✅ SHIPPED 2026-04-21
**Phases:** 21–27
**Total Plans:** 24

## Overview

The Pre-Beta Refactoring milestone cleaned up the technical debt accumulated during the v0.4 Functional Alpha sprint, restored a broken production state, and executed a significant architecture pivot. When this milestone began, commit `6cfb0d3` had wiped the security pipeline and Discord interface from the codebase. When it ended, the system had 131 passing tests, full Nyquist compliance across all shipped phases, and a new architecture: Pi removed from the base stack, LiteLLM-direct AI calls, and a module API gateway as the extensibility model.

## Milestone Summary

### Key Decisions

| Decision | Rationale | Outcome |
|----------|-----------|---------|
| Path B: Pi demoted to optional profile | Architecture crisis — Pi was bypassed in Phase 25 message route; full replan needed | Pi removed from base stack; module gateway is the extensibility model |
| LiteLLM-direct for all AI calls | Eliminates DUP-02/03 (dual provider abstraction layers); single unified interface | DUP-01–05 all eliminated |
| shared/sentinel_client.py canonical client | DUP-04: call_core() inline in both interfaces; shared package eliminates duplication | All interfaces import SentinelCoreClient |
| buildApp() factory pattern for Fastify | Canonical Fastify testing pattern; separates construction from port binding | vitest can test bridge.ts routes via app.inject() |
| Module API gateway contract | Path B reference implementation for v0.5 Pathfinder module | POST /modules/register + proxy pattern established |
| SecondaryClassifier callable abstraction | OutputScanner was hardcoded to Anthropic SDK; violates AI-agnostic design | Generic callable; provider-agnostic |
