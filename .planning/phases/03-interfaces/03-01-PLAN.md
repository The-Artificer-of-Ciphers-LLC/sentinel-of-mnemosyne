**Step 1 — Add middleware to sentinel-core/app/main.py:**

Add these imports at the top of main.py (after existing imports):
```python
from starlette.middleware.base import BaseHTTPMiddleware
```

Add this class definition BEFORE the `app = FastAPI(...)` line:
```python
class APIKeyMiddleware(BaseHTTPMiddleware):
    """Require X-Sentinel-Key header on all non-health endpoints (IFACE-06)."""

    async def dispatch(self, request, call_next):
        if request.url.path == "/health":
            return await call_next(request)
        key = request.headers.get("X-Sentinel-Key", "")
        if key != settings.sentinel_api_key:
            return JSONResponse({"detail": "Unauthorized"}, status_code=401)
        return await call_next(request)
```

After the `app = FastAPI(...)` block and BEFORE `app.include_router(message_router)`, add:
```python
app.add_middleware(APIKeyMiddleware)
```

The resulting order in main.py must be:
1. `app = FastAPI(..., lifespan=lifespan)`
2. `app.add_middleware(APIKeyMiddleware)`
3. `app.include_router(message_router)`
