## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| PNACORSMiddleware subclass | Subclassing Starlette CORSMiddleware to inject PNA header injected it on ALL responses, not just OPTIONS preflight — spec violation. Starlette 1.0 supports `allow_private_network=True` natively. | advisory | Always check if the framework already supports a feature before wrapping it. Read the installed version's `__init__` signature first. |
| allow_origins wildcard | Starlette does exact-string matching on `allow_origins`; `*.forge-vtt.com` was silently ignored. | advisory | Use `allow_origin_regex` for any pattern that requires glob/regex matching. |
